I code things for fun. I started with Java in order to make Bukkit/Spigot plugins and Forge mods. I've also made some desktop programs, such as mauCapture. More recently I've used Golang, Python and JavaScript. All my public projects can be found on GitHub. See the Contact section for my contact information.


Some of my work:

gomuks

A terminal based Matrix client written in Go. Aims to support as many Matrix/Riot features as possible in modern terminal emulators.

mautrix-telegram

A hybrid puppeting/relaybot Matrix-Telegram bridge to allow you to use Telegram from Matrix. Written in Python.

goPacked

A CLI Minecraft modpack manager written in Go. goPack definitions use a fairly simple JSON format and can be hosted anywhere.
